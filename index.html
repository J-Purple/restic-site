Restic + Rclone Backup System

Restic + Rclone Backup System is a small tool that helps you configure and run backups and synchronisation tasks using restic and rclone.

It is intended for users who want a simple way to manage backup jobs to local or remote storage while keeping control over their own infrastructure.

This site provides general information about the project, along with links to the privacy policy and terms of service for the application.
